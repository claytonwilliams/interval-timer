Visualizing UW Prerequisite Course Sequences

An interactive tool that allows a user to explore course prerequisite sequences across the University of Washington course catalog.

  • Final Project for Jeff Heer's CSE 512 Data Visualization course in spring 2016.
  • Developed using the JavaScript library D3.js.
  • User inputs a course in the text field and a dendrogram is created with the course's prerequisites as the tree children.
